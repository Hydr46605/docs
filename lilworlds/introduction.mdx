## What is LilWorlds?

LilWorlds is a **performant, modular world management plugin** for Minecraft servers that provides comprehensive world management capabilities. Designed as a modern alternative to Multiverse-Core, LilWorlds focuses on performance, organization, and modularity.

## Quick Overview

LilWorlds provides two main command categories:

- **`/world` (or `/w`)** - World management commands
- **`/worlds`** - Plugin administration commands

### Example Commands

```bash
# Create a new world
/world create myworld

# Create with custom generator
/world create myworld NORMAL -g VoidGenerator

# Clone an existing world
/world clone spawn spawn_backup

# List all worlds
/world list

# Delete a world (with confirmation)
/world remove myworld
```
